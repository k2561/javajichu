			第十四章带参数的方法
一、定义带参数的方法：
	访问修饰符  返回值类型  方法名(参数类型 参数名，参数类型2 参数名2){
		//方法体
	}
	eg:public String zhazhi (String shuiguo){
		return "水果汁";
	}
二、调用带参的方法：
	1.传入的参数必须要和定义时方法的参数一致。
	2.传入的参数类型必须要和定义时的类型一致。
	3.传入的参数顺序必须要和定义时的顺序一致。
三、调用带参数方法的语法：
	1.在同一个类中：
		方法名(参数列表)
	2.在不同的类中：
		先创建方法所在类的对象，然后对象名.方法名（参数列表）
四、形参：定义方法时的参数
五、实参：调用方法时传入的参数
六、调用方法时注意的事项：
	1.调用方法时不能指定实参类型
	2.形参和实参的数据类型不一致
	3.形参和实参的数量不一致
	4.调用方法后没有对返回值做任何处理。
七、数组也可以作为参数传进去
	eg:
	public void show(int [] name){
		
	}
	
	调用方法：
	int name [] = new int[5];
	show(name);
八、对象也可以作为参数传递进去：
	eg:
	Student stu = new Student();
	//定义方法
	public void getAvg(Student stu){
		//方法体
	}
	//调用方法
	getAvg(stu);
